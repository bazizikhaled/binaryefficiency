Bookmarks Manager:

The Bookmarks Manager script is a command-line tool for organizing bookmarks, links, texts, and PDFs. It uses dmenu for dynamic menus, supports easy addition, categorization, and deletion of bookmarks, and offers clipboard integration for quick content saving. The script is user-friendly and customizable
